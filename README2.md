# Introduction à Git & GitHub

## 1. Titre & Description

Ce projet présente les bases de Git et GitHub.

Git est un système de gestion de versions qui permet de suivre les modifications du code. GitHub est une plateforme qui permet d’héberger et de collaborer sur des projets.

---

## 2. Prérequis

Avant de commencer, vous devez installer :

- Git → https://git-scm.com/install
- VS Code (ou autre éditeur)
- Un compte GitHub

---

## 3. Installation

Cloner un dépôt :

git clone git clone https://github.com/Balkiss678/git-initiation

Configurer Git :

git config --global user.name "Balkiss Mzoughi"

git config --global user.email "balkissexrd@gmail.com"

4. Utilisation 
- Ajouter les fichiers :
git add .

- Créer un commit :
git commit -m "Premier commit"

- Envoyer vers GitHub :
git push origin main

- Récupérer les modifications :
git pull
